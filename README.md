# Tech-Blog

## Description

This application provides a platform for tech bloggers the ability to post and comment on other blog posts. Bloggers can signup, login, create and comment on posts. It uses Express.js for middle and backend, Handlebars for frontend, mysql as the database, bcrypt to encrypt user passwords and bulma for styling.

## Installation

- Clone this repo
- Run `npm i` to install dependencies
- Update `.env` with your MySQL Workbench credentials

## Usage

- Run `npm start` to start the server
- Click the sign up button and enter your credentials
- Once logged in, Create a blog post or comment on other blog posts
